1. Build

The first step is to launch PhpMetrics during build.

To do this, select "Run Shell Script" and insert the following code in the text box that appears:

{% highlight bash %} #!/bin/bash mkdir -p build wget https://github.com/Halleck45/PhpMetrics/raw/master/build/metrics.phar php metrics.phar --report-html=build/phpmetrics.html --report-xml=build/phpmetrics.xml --violations-xml=build/violations.xml {% endhighlight %}
