PhpMetrics
Static analysis tool for PHP

Easy to use

Download PhpMetrics (with composer): 

php composer.phar global require 'halleck45/phpmetrics=@dev'

If you're installing a global package with Composer for the first time, remember to add the following line to your `.bashrc` file: `export PATH=~/.composer/vendor/bin:$PATH`

Usage: 

vendor/bin/phpmetrics --report-html=/path/of/report.html <folder or filename>

PhpMetrics come also as [Phar archive](https://github.com/Halleck45/PhpMetrics/raw/master/build/metrics.phar).

Various metrics

  • Maintenability Index
  • Cyclomatic complexity
  • Halstead's metrics
  • Lack of cohesion of methods
  • Abstraction / Instability
  • ...
  • and more !
