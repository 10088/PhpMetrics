PhpMetrics
Static analyze tool for PHP

Easy to use

Download PhpMetrics: 

wget https://github.com/Halleck45/PhpMetrics/raw/master/build/metrics.phar

Use it: 

php metrics.phar --report-html=/path/of/report.html <folder or filename>

Various metrics

  • Maintenability Index
  • Cyclomatic complexity
  • Halstead's metrics
  • Lack of cohesion of methods
  • Abstraction / Instability
  • ...
